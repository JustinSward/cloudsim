How to Use CircuitSimulatorApp

CircuitSimulator App can simulate electronic circuits given a netlist. A netlist is a description of the parts in the circuit and their interconnections.

Importing a Netlist

You can import a netlist from a .net file either previously exported from CircuitSimulator App or from other programs that use the ExpressPCB netlist file format (used by popular software such as Analog Device's LTSpice).

Importing a netlist will clear the current netlist so be sure to save your state if you would like to be able to return to it by exporting it first.

Building a Netlist

The netlist can be built by adding components:

  1. Select a component type from the drop-down menu (ie. Resistor)
  2. Enter an identifier for this device (ie. "5", "source", etc)
  3. Enter the numerical value in base units
  4. Enter the first node connected to the device (positive, if polarity sensitive)
  5. Enter the second node connected to the device (negative, if polarity sensitive)
  6. Click the "Add Component" button to add the part to the netlist

Components in the list can be removed from the netlist by clicking the "Delete" link under the "Action" column for that part.

You can save your current netlist by using the "Export Netlist" button, which will provide a .net netlist file compatible with the ExpressPCB format. This can be imported back into CircuitSimulator App or into software such as LTSpice.

Setting up the Simulation

The simulation parameters must be set prior to the simulation.

  1. Select a simulation type from the drop-down menu:
  2. Enter a numerical node value to simulate
  3. Enter the starting (low) and end (high) frequencies, if performing a Frequency analysis
  4. Click "Run Simulation" to run the simulation with the given settings.
